Amazon Clone

This project involved building a fully functional e-commerce website frontend, replicating core features of Amazon. This project served as a valuable learning experience, solidifying my frontend development skills and preparing me to actively contribute to our client's ongoing e-commerce website development. To enhance user experience, I implemented several key functionalities:

  • Unique Order IDs: Each order generates a unique ID, ensuring proper tracking and management.

  • Order Tracking and History: Users can easily track placed orders, view their history, and access details from the dedicated "Returns and Orders" section.

ECP and ABAP Implementation

Client: Beam Consolidated

This project focused on optimizing the payroll process for the organization using SAP technologies. Here's what I achieved:

  • ECP and ABAP on HANA Implementation: Successfully implemented SAP Enterprise Central Component (ECC) and ABAP programming language on the HANA database platform. This enhanced performance and efficiency for core business processes.
  • Automated Payslip Generation: Developed a system that automatically generates payslip PDFs for each employee, streamlining the process and reducing manual effort.
  • Monthly Whatsapp Delivery: Integrated a system to deliver payslips directly to employees' WhatsApp accounts every month, ensuring timely and convenient access to their pay information.
  • Third-Party Integration via RESTful APIs: Created secure RESTful APIs (web services) to seamlessly transfer payroll data to a third-party portal, enabling efficient data exchange.
  • QR Code Generation and Mobile App Integration: Designed a program that generates QR codes on payslips. Scanning these codes redirects users to dedicated Android and iOS applications (details can be mentioned if applicable).
  • RICEF Development: Developed and maintained essential RICEF (Reports, Interfaces, Conversions, Enhancements, and Forms) elements to support daily operations and user needs within the SAP landscape.

Project Impact:

  • Increased efficiency and accuracy in payroll processing.
  • Improved employee experience with timely and convenient payslip access.
  • Enhanced data exchange with third-party systems.
  • Streamlined daily operations through RICEF development.
